Installation of How2Say

The procedure to perform a standard installation of How2Say on a dedicated server running Windows is described here.

Server Specifications

The server required to run How2Say should comply with the following specifications.

How2Say must be deployed on a dedicated server, which can be either a physical or a virtual machine. Please note that the server needs to be 64-bit.

CPU:

One CPU (Quadricore), 64-bit

RAM:

32 to 64 Gb

Operating system:

  • Windows Server 2008 or Windows 7, 64-bit version
  • or GNU/Linux Ubuntu 12.04 LTS, 64-bit version
Applications:

The applications to be installed on it will be the following:

  • Java JRE 6 or 7
  • Apache Tomcat 7.0 (to publish the application online)
  • How2Say (Java classes distributed in a .war file through the Tomcat)

How2Say is available both for Windows (tested on Windows 7 and Windows 2008 Server) and for GNU/Linux (tested on Ubuntu 12.04 LTS).
