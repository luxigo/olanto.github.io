Release Roadmap

Olanto plans to develop and distribute various CAT tools according to the following roadmap:

  • Sept 2016: myTMX, a concordancer on TMX
  • june 2016: How2Say, a new release
  • April 2016: myTerm 3.0, adding images
  • March 2016: myCAT, a new release

Also planned: a cross-lingual search tool, an EDMS / Translation Management tool, etc... All suggestions are welcome!
