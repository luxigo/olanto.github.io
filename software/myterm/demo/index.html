Interface language !

We have a dynamic approach to integrate new languages for the Graphical User Interface of myTerm. We have all the interface's components in two languages (en, fr). If you think that you can add a new language for the interface please contact us at jacques [at] olanto.org (contact) and we will guide you for its integration in the tool.

We imported terminologies from different sources: 

  • IATE: 24 languages, 8.8 Mo terms, origin:
    • http://iate.europa.eu/about_IATE.html
  • FISH_NAME: (EN, ES, FR, LA) 8'000 terms, origin FAO:
    • http://www.ehu.eus/PAT/Glosarios/FISHTERM_94.txt
  • MED_UTX: (EN, JA) 54'000 terms, origin Kitazato University:
    • http://cpansearch.perl.org/src/BYUTRG/Convert-TBX-UTX-0.032/t/Corpus/Sample.utx
