Installation de How2Say

La procédure permettant d'effectuer une installation classique de How2Say sur un serveur dédié équipé de Windows est décrite ici (en anglais).

Spécifications du serveur

Le serveur qui hébergera How2Say doit répondre aux spécifications suivantes (en anglais).

How2Say doit être déployé sur un serveur dédié, qui peut être soit une machine physique ou une machine virtuelle. Veuillez noter que ce serveur doit être en 64 bits.

CPU:

Un CPU (Quadricore), 64-bit

RAM:

32 à 64 Gb

Système d'exploitation:
  • Windows Server 2008 ou Windows 7, version 64-bit
  • ou GNU/Linux Ubuntu 12.04 LTS, version 64-bit
Applications:

Les applications à installer sont les suivantes:

  • Java JRE 6 or 7
  • Apache Tomcat 7.0 (pour publier l'application en ligne)
  • How2Say (Classes java distribuées dans un fichier .war dans la Tomcat)

How2Say est disponible aussi bien pour Windows (testé sur Windows 7 et Windows 2008 Server) que pour GNU/Linux (testé sur Ubuntu 12.04 LTS et 14.04 LTS)
