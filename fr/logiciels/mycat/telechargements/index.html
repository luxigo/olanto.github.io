Cartes d'alignement additionnelles

La version de base de myCAT est livrée avec les cartes d'alignement suivantes: anglais-arabe, anglais-espagnol, anglais-français, anglais-portugais et anglais-russe. Il est possible d'ajouter les paires de langues suivantes :

  • anglais-allemand
  • anglais-chinois

Veuillez lire la documentation : Ajouter des Cartes d'alignement

myCAT est disponible aussi bien pour Windows (testé sur Windows 7 et Windows 2008 Server) que pour GNU/Linux (testé sur Ubuntu 12.04 LTS)
